EndType

Del.» type TEndType = (etClosed, etButt, etSquare, etRound);

C++ » enum EndType { etClosed, etButt, etSquare, etRound };

C#  » public enum EndType { etClosed, etButt, etSquare, etRound };


The OffsetPolyLines function has an EndType parameter that accepts one of four end types:

  • etClosed: Ends are joined using the JoinType style.
  • etButt: Ends are squared off abruptly.
  • etSquare: Ends are squared off at delta units
  • etRound: Ends are rounded off at delta units
